Project Overview

This project implements a 16x8 bit synchronous FIFO (First-In-First-Out) memory and its complete verification environment 
using SystemVerilog. 

The verification environment is built using Object-Oriented Programming (OOP) concepts and follows a modular architecture.

- FIFO Module
Depth: 16 locations

Data Width: 8 bits

Control Signals:

clk: Clock input

rst: Reset (active high)

wr: Write enable

rd: Read enable

full: FIFO full flag

empty: FIFO empty flag
